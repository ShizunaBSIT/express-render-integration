# Git commands

### add files to be updated
<p>```git add .``` </p>
<p>```git add {filename}``` </p>


### add commit message
<p> ```git commit -m "{message}"```</p>


### Pull from repository:
<p> ```git pull origin {branch name}``` </p>


### Push to repository:
<p> ```git push origin {branch name}``` </p>
